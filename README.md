# FullStack

One of my first projects. Built with friends while we were figuring out how programming works.

It's an ecommerce project — we were learning full stack development by building something real together. The code reflects that learning process.

## What we built

- Basic ecommerce functionality
- Frontend and backend working together
- A collaborative workflow using feature branches and pull requests

## Notes

This was about learning, not shipping. The code is rough in places — that's the point.
